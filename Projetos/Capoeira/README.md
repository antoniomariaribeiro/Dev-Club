# Capoeira Nacional

Projeto web para gestão e apresentação do grupo de Capoeira Nacional.

## Estrutura
- `backend/`: API Node.js/Express/MongoDB
- `public/`: Frontend (HTML, CSS, JS, imagens)

## Como rodar
1. Instale as dependências em `backend/`
2. Inicie o backend: `node backend/index.js`
3. Acesse os arquivos HTML em `public/`
